What is this hub about?

This site provides a community-driven data repository and curation service for molecular structures, models, therapeutics, and simulations related to computational research on the SARS-CoV-2 / COVID-19 / "Coronavirus" pandemic. For more info about this project, see the about page. We gladly accept any of the types of data hosted on the website (and others if you propose new types). For instructions on how to contribute, please see the contributing page.
