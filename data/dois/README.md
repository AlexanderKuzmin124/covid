# Metadata for DOIs issued by MolSSI

## To add new resources

Contribute a separate [YAML file](https://yaml.org/) for each entry.

### YAML schema

Each entry has the following required and optional keys:
```
name: (required)
url: (optional)
doi: (required)
description: (required)
owner: (required)
creator: (required)
content: (required)
purpose: (required)
file_types: (required)
files: (required)
size: (required)
license: (optional)
```
